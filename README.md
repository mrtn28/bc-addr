### 🏠 [Homepage](https://github.com/mrtn28/bc-addr#readme)

## Install

```sh
npm install
```

## Usage

Create bitcoin mnemonic:

```sh
npx bc-addr
```

Generate addresses from zpub:

```sh
import { zpubToAddress } from 'bc-addr'
```

## Run tests

```sh
npm run test-zpub-to-address
```
